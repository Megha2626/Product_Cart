This is a simple front-end dessert shop project built using HTML, CSS, and JavaScript. Users can browse dessert cards, add items to a cart, view selected items, see the total price update dynamically, and confirm their order.

## Features
- Responsive product grid layout
- Hover effects on product cards
- Add to Cart functionality
- Dynamic cart item list
- Automatic total price calculation
- Cart count update
- Confirm Order with cart reset

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, Media Queries)
- Vanilla JavaScript (DOM Manipulation, Events)

## Learning Outcomes
- DOMContentLoaded usage
- Event handling
- Working with datasets
- Managing application state using arrays
- Updating UI dynamically

## How to Run
1. Clone or download the project
2. Open `index.html` in your browser
3. Click "Add to Cart" to test functionality
